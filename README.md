# 💌 Dear Me - Android App

Dear Me is a beautifully designed Android application built using Jetpack Compose. It allows users to read motivational "letters to self" in an interactive, clean, and visually appealing format.

---

## ✨ Features

- 📜 Scrollable list of daily letters  
- 💌 Expand/Collapse functionality for each letter  
- 🖼️ Image-based card UI  
- 🎨 Modern Material 3 design  
- ⚡ Smooth animations using animateContentSize  
- 📱 Fully built using Jetpack Compose  

---

## 🛠️ Tech Stack

- Language: Kotlin  
- UI Toolkit: Jetpack Compose  
- Design: Material 3  
- IDE: Android Studio  

---

## 📂 Project Structure

com.example.dearme

- MainActivity.kt -> Entry point of the app  
- DearMeApp() -> Main UI layout  
- LetterCard() -> Reusable card component  
- Letter Model -> Data structure for each letter  
- getLetters() -> Provides list of letters  
- Theme -> Custom Material theme  

---

## 🚀 How It Works

- The app loads a list of letters using getLetters()  
- Each letter is displayed inside a card layout  
- Users can tap on a card to:  
  - Expand and read the full message  
  - Collapse it back
